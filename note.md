## REACT INSTALLATION

- run `npx create-react-app@5 pizza-menu` '5' means version 5.0
- latest version use korar jonno run `npx create-react-app@latest pizza-menu`

## BUILD RUN

- run `npm run start/npm start`
- run `npm run build`

## HOW THE SYSTEM WORK

- all the assets go to the public folder
